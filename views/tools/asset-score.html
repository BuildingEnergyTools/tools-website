The Building Energy Asset Score (Asset Score) is a free, web-based tool for assessing the physical and structural energy efficiency of commercial and multifamily residential buildings.

Asset Score assesses the energy efficiency of building assets (including envelope, mechanical, electrical and service hot water systems) and generates an Energy Asset Score Report which includes:

  • An overall efficiency score ranging from 1-10
  • An energy efficiency assessment of the building’s individual systems
  • Total estimated building energy usage under standard operation and by end use
  • Opportunities to improve building efficiency, and a “potential” energy efficiency score based on identified upgrades

Because Asset Score reflects the energy efficiency of a building based solely on its design, construction, and energy systems, users can:

  • Quickly compare the physical energy efficiency of their building with other buildings
  • Communicate building efficiency to the marketplace
  • Identify opportunities to invest in energy efficiency upgrades

Getting Started

To get started, register for an Asset Score account. View the Introduction to Asset Score webinar on this page and review the additional resources on this page as well as the Asset Score User Guide.

An ENERGY STAR Portfolio Manager score enables the comparison of buildings based on their energy consumption, as indicated by actual energy bills (how a building actually performs based on use). The Portfolio Manager score may reflect how efficiently a building is operated and maintained.

Asset Score reflects the energy efficiency of a building based solely on its design, construction, and energy systems (how a building should perform based on construction). Think of it as a “miles per gallon” sticker for buildings.

Using Asset Score and Portfolio Manager together provides powerful information that can identify energy upgrades and improvements in building operations. For example, in the graphic above:

  • A building that receives a low ENERGY STAR score and a high Asset Score may benefit from a re-tuning or retrocommissioning.
  • A building that receives a high ENERGY STAR score and a low Asset Score may benefit from some capital improvements.

The Seattle Building Tune-Up Accelerator Program recently used both tools in this way; their website includes information on the program, including a link to a recorded webinar and a case study.

Goals & Outcomes

Use Asset Score to identify cost-effective energy efficiency measures to help buildings deliver EUI and GHGe metrics for compliance with a building performance policy.

Links for Developers

We welcome use of the Asset Score by energy services companies, consultants, software providers and other parties that offer services to building owners to improve the energy efficiency of buildings. Such providers can integrate their tools seamlessly with the Asset Score through an application programming interface (API).

RESTful Web Service

The Asset Score API is a RESTful style web service. For more information about REST, please visit: http://en.wikipedia.org/wiki/Representational_state_transfer

API Organization Token

In order to create an account to authenticate and utilize the API, users must first get an API Organization Token. The token must be passed along with the parameters used when creating a new account. Please send an email with the subject line "Request API Token" to: asset.score@ee.doe.gov

Once the organization token is received the API can be utilized to create a user account. A user account is used to obtain an API Access Token, described below. Follow the API documentation for details of the parameters needed: POST /api/v2/users

API Access Token

Once a user account has been created the user can authenticate against the API to generate a user-specific access token. All subsequent interactions with the API must include the authenticated user's API token. Follow the API documentation for details of authentication method: POST /api/v2/users/authenticate

The API Access Token is specific to an individual user, and is distinct from the API Organization Token. The API Organization Token is used only to create user accounts and to authenticate those accounts in order to receive the API Access Token.

To use your API access token, simply pass the key as a URL parameter when making requests. For example:

GET https://api.labworks.org/api/v2/preview_buildings?token=YOUR_UNIQUE_TOKEN

API Documentation

All API documentation can be found at: API Documentation

Summary of Steps

  • Request an API Organization Token. Send email with subject line "Request API Token" to: asset.score@ee.doe.gov
  • Use the API Organization Token to create a user account: POST /api/v2/users
  • Authenticate the user account using the API Organization Token to receive an API Access Token: POST /api/v2/users/authenticate
  • Use the API Access Token in further API requests.
