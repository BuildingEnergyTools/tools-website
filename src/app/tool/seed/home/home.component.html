The Standard Energy Efficiency Data (SEED) Platform is a central database for building-related data. It merges information from ENERGY STAR Portfolio Manager, Audit Template, and other sources in one place, behind one intuitive interface. With just a few clicks, you can:

  • Merge datasets
  • Assess the completeness of audit reports
  • Perform automated quality assurance
  • Share data with third parties
  • Track the progress of BEPS program compliance
  • Leverage analysis & evaluation tools to identify upgrade opportunities within buildings and across the city’s building stock, assess BEPS program performance, and set program cycle goals
  • Share data with Customer Relationship Management (CRM) systems and dashboards.

Because SEED automates so many manual, time-intensive processes, even organizations with severely limited resources and staff can use it to achieve their program goals.

Sample Data

The SEED Sample Data collection includes two files and can be downloaded from the table below:

  • covered-buildings-sample.csv—sample tax assessor data with 512 building records
  • portfolio-manager-sample.csv—sample Energy Star Portfolio Manager data with 512 records which match the data in covered-buildings-sample.csv based on the Address field

Note that this represents an ideal case, but it can be used to confirm proper program performance.

All sample data should work with all versions of SEED, although the newer datasets may contain fields that do not exist in earlier versions.
